Image Based Rendering

Geometric Modelling presented some major disadvantages from being labour intensive to time consuming. Thus, I used the 3D warping technique. In this, several 2-D images were used and after mapping the corresponding pixels in these images Point Cloud Library was used to create their 3-D model in opencv using python.
